Rock-Paper-Scissor.py is a python file

It is Played between Computer and Human(user)

Firsty, It asks for User Name Then Start The Game

There Are Total 5 Rounds to Play If done Then Asks for user to play again if user entered 'y' it continues else entered any letter including 'n' the game is exited

In 5 Rounds, If A tie Occur it may not consider as a round, that is, only win is counted as a round

At Begining as well as after typing 'y' to play again the display gets clear to look Interactive
