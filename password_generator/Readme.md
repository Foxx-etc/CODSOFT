The password_generator.py is a python file to generate random password.

It takes two Input from user which ask for length and complexity(easy, medium, hard)
Both The Input goes through validation, that is, if user typed any wrong Input It gives message until the user entered the correct one and then moved to further!

Password Length must be greater than 6
The password is generated based on : 
-lower case alphabet letters
-upper case alphabet letters
-symbols
-numbers
